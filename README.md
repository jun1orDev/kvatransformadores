# kvatransformadores
Código para o site institucional da KVA Transformadores.

Documentação da migração e da arquitetura em [docs/INDICE_DOCUMENTACAO.md](docs/INDICE_DOCUMENTACAO.md).

## Como rodar o projeto

Requisitos:

- Node.js 18 ou superior

Instalação e execução:

1. Instale as dependências com `npm install`
2. Inicie o ambiente de desenvolvimento com `npm run dev`
3. Gere a build de produção com `npm run build`
4. Visualize a build localmente com `npm run preview`
